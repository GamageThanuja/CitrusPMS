# Hotel Mate Dashboard

Hotel Mate is a comprehensive hotel management system dashboard built with Next.js and Tailwind CSS. It provides a complete solution for managing hotel operations including reservations, room management, point of sale, financials, and more.

## Features

- **Dashboard Overview**: Get a quick glance at key metrics and performance indicators
- **Reservation Management**: Handle front desk operations, bookings, arrivals, and departures
- **Room & Rate Management**: Manage room types, rates, and distribution channels
- **Point of Sale (POS)**: Process food, beverage, and service sales
- **Financial Management**: Track purchases, expenses, payables, receivables, and profit/loss
- **Settings**: Configure users, currency, taxes, travel agents, and suppliers
- **Multi-language Support**: Built-in internationalization
- **Theme Support**: Light, dark, and custom color themes
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Prerequisites

Before you begin, ensure you have the following installed!:
- [Node.js](https://nodejs.org/) (v18 or newer)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) or [pnpm](https://pnpm.io/)
- [Git](https://git-scm.com/)
